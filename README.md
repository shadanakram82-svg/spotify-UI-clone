# Spotify UI Clone 🎵

A responsive frontend clone of Spotify built using pure HTML and CSS.  
This project replicates the layout, grid system, hover interactions, and responsive behavior of Spotify’s web interface.

---

## 🔗 Live Demo

👉 https://spotify-ui-01.netlify.app/

---

## 📌 Project Overview

This project focuses purely on frontend development.  
It recreates Spotify’s modern UI using:

- CSS Grid for layout
- Flexbox for alignment
- Hover effects for interactive UI
- Media queries for responsiveness

No JavaScript or backend is used in this version.

---

## ✨ Features

- Fully responsive layout
- Grid-based music card section
- Sidebar navigation
- Sticky header
- Hover animation effects on cards
- Clean and modern UI design
- Using media queries

---

## 🛠 Tech Stack

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Media Queries
- Netlify (Deployment)
